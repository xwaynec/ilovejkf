I love jkforum
==========


Inspired by iloveck101


How to install
==========

```bash
pip install ilovejkf
```


How to use
==========

```bash
ilovejkf [jkforum url]
```


Example

```bash
ilovejkf https://www.jkforum.net/thread-xxxxx-1-1.html
```


You could find account folder in ~/Pictures/`ilovejkf`

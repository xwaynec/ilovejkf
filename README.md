I love timliao
==========


Inspired by iloveck101


How to install
==========

```bash
pip install ilovejkf
```


How to use
==========

```bash
ilovejkf [timliao url]
```


Example

```bash
ilovejkf http://www.timliao.com/bbs/viewthread.php?tid=xxxxx
```


You could find account folder in ~/Pictures/`ilovejkf`
